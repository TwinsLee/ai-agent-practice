# AI智能体实战 | AI Agent Practice

> AI 智能体实战笔记。踩坑、反思、方法论。

## 关于这个仓库

这是我在实际工作中深度使用 AI 智能体（主要基于 [OpenClaw](https://github.com/openclaw/openclaw)）的经验记录。

关注的不是"AI 能做什么"，而是**真正把 AI 智能体用到日常工作中会遇到什么问题、怎么解决**——安全、记忆、架构、成本、多人协作、组织推广……这些在官方文档和教程里很少讲到的东西。
